## HackerNews Clone

<p>Welcome to the HackerNews Clone Repo</p>

### Get started

```bash
git clone https://github.com/Sachin-chaurasiya/HackerNews-Clone.git

```

#### Install Dependencies

```bash
cd HackerNews-Clone
npm install
```

#### Spin up the server

```bash
npm start
```
